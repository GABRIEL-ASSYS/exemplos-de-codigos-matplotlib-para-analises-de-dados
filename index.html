📐 Matplotlib: Criar Figuras e Eixos

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar figura e eixos com tamanho e resolução personalizados
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# 2. Exemplo de plot
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]
ax.plot(x, y, marker='o')

# 3. Rótulos e título
ax.set_title('Exemplo de Gráfico')
ax.set_xlabel('Eixo X')
ax.set_ylabel('Eixo Y')

# 4. Exibir
plt.show()

🧠 Para que serve?

Permite criar o canvas de desenho com dimensões (em polegadas) e qualidade (dpi) controladas, garantindo figuras consistentes em relatórios e apresentações.

🔍 Explicação passo a passo

  • plt.subplots(figsize=(10, 6)): define largura=10" e altura=6" da figura.
  • dpi=100: especifica 100 pontos por polegada, ajustando a resolução.
  • fig, ax: retorna objetos Figure e Axes para personalizar completamente o plot.
  • ax.plot(x, y): desenha a curva no eixo.
  • ax.set_title/xlabel/ylabel: adiciona texto descritivo ao gráfico.
  • plt.show(): apresenta a figura na tela ou em notebooks.

🧾 Exemplo de Saída

Uma figura de 10×6" com resolução 100 dpi contendo um gráfico de linha simples, pontos nos valores e rótulos claros nos eixos.

📈 Gráfico de Linha

🐍 Exemplo Python

import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(10, 6), dpi=100)
ax.plot(df['data'], df['vendas'], marker='o', linestyle='-')
ax.set_title('Vendas ao Longo do Tempo')
ax.set_xlabel('Data')
ax.set_ylabel('Vendas')
plt.show()

🧠 Para que serve?

Exibe tendências ao longo do tempo, ideal para séries temporais e análise de evolução de métricas.

🔍 Explicação passo a passo

  • fig, ax = plt.subplots(...): cria a figura e o eixo de desenho.
  • ax.plot(df['data'], df['vendas'], marker='o', linestyle='-'): plota vendas x data com marcadores e linhas sólidas.
  • ax.set_title(...): define o título do gráfico.
  • ax.set_xlabel/ylabel: rotula os eixos X e Y.
  • plt.show(): renderiza a figura.

🧾 Exemplo de Saída

Gráfico de linha mostrando a evolução de vendas no tempo, com pontos destacados em cada data.

📦 Boxplot

🐍 Exemplo Python

import matplotlib.pyplot as plt

fig, ax = plt.subplots(figsize=(10, 6), dpi=100)
grupos = [grupo['valor_total'] for _, grupo in df.groupby('categoria')]
ax.boxplot(grupos)
ax.set_xticklabels(df['categoria'].unique(), rotation=45)
ax.set_title('Boxplot de Valor Total por Categoria')
ax.set_ylabel('Valor Total')
plt.show()

🧠 Para que serve?

Resume estatísticas de dispersão (mediana, quartis, outliers) por categoria de forma compacta.

🔍 Explicação passo a passo

  • [grupo['valor_total'] for _, grupo in df.groupby('categoria')]: cria lista de séries para cada categoria.
  • ax.boxplot(grupos): desenha boxplots lado a lado.
  • ax.set_xticklabels(...): rotula categorias no eixo X.
  • ax.set_title/xlabel/ylabel: adiciona título e rótulos.
  • plt.show(): exibe o gráfico.

🧾 Exemplo de Saída

Boxplots comparativos por categoria, mostrando mediana, quartis e potenciais outliers.

🔲 Subplots e Layouts

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar grid de subplots 2x2
fig, axes = plt.subplots(nrows=2, ncols=2, figsize=(12, 8), dpi=100)

# 2. Plotar em cada eixo
axes[0, 0].plot(df['data'], df['vendas'], marker='o')
axes[0, 0].set_title('Vendas ao Longo do Tempo')

axes[0, 1].hist(df['preco_unitario'], bins=20)
axes[0, 1].set_title('Distribuição de Preço')

axes[1, 0].bar(df['categoria'], df['valor_total'])
axes[1, 0].set_title('Valor Total por Categoria')
plt.setp(axes[1, 0].get_xticklabels(), rotation=45)

axes[1, 1].scatter(df['quantidade'], df['valor_total'], alpha=0.7)
axes[1, 1].set_title('Quantidade vs. Valor Total')

# 3. Ajustar espaçamentos
fig.tight_layout()

# 4. Exibir
plt.show()

🧠 Para que serve?

Organiza múltiplos gráficos em uma grade, permitindo comparar diferentes visualizações de dados lado a lado de forma clara e consistente.

🔍 Explicação passo a passo

  • plt.subplots(nrows=2, ncols=2, figsize=(12, 8), dpi=100): cria uma figura com 4 eixos dispostos em 2 linhas e 2 colunas, de tamanho 12×8" e 100 dpi.
  • axes[i, j].plot/hist/bar/scatter: acessa cada Subplot (linha i, coluna j) para desenhar o gráfico correspondente.
  • axes[...].set_title(...): define título individual para cada gráfico.
  • plt.setp(axes[1,0].get_xticklabels(), rotation=45): rotaciona rótulos do eixo X no gráfico de barras para melhor leitura.
  • fig.tight_layout(): ajusta automaticamente margens e espaçamentos entre subplots.
  • plt.show(): renderiza toda a figura com a grade de gráficos.

🧾 Exemplo de Saída

Uma matriz 2×2 de gráficos: linha de séries temporais, histograma de preços, gráfico de barras por categoria e scatter de quantidade vs. valor, com títulos e espaçamento bem ajustados.

🎨 Personalização de Estilos

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar figura e eixos
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# 2. Dados de exemplo
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# 3. Personalização de estilo
ax.plot(x, y, color='C1', linestyle='--', marker='s', label='Série A')

# 4. Título, legenda e exibição
ax.set_title('Linha Personalizada')
ax.legend()
plt.show()

🧠 Para que serve?

Permite ajustar cores, estilos de linha e formatos de marcador para diferenciar séries e reforçar a comunicação visual dos dados.

🔍 Explicação passo a passo

  • fig, ax = plt.subplots(...): cria a figura e os eixos para plotagem.
  • x, y: listas de coordenadas para o gráfico.
  • color='C1': escolhe a cor (segundo tom padrão do Matplotlib).
  • linestyle='--': define linha tracejada.
  • marker='s': usa marcador em forma de quadrado.
  • label='Série A': adiciona um rótulo à série para a legenda.
  • ax.legend(): exibe a legenda no gráfico.
  • plt.show(): renderiza a figura na saída.

🧾 Exemplo de Saída

Gráfico de linha com cor secundária, linha tracejada e marcadores quadrados, acompanhado de uma legenda identificando a série.

✒️ Títulos, Rótulos e Anotações

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar figura e eixos
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# 2. Exemplo de dados
x = [1, 2, 3, 4, 5, 6]
y = [8000, 9500, 12000, 14000, 15000, 13000]

# 3. Definir título e rótulos
ax.set_title('Vendas Mensais')
ax.set_xlabel('Mês')
ax.set_ylabel('Faturamento (R$)')

# 4. Anotar um ponto específico
ax.annotate(
    'Pico de vendas',
    xy=(5, 15000),
    xytext=(6, 18000),
    arrowprops=dict(arrowstyle='->')
)

# 5. Plotar linha e exibir
ax.plot(x, y, marker='o', linestyle='-')
plt.show()

🧠 Para que serve?

Adiciona contexto e clareza ao gráfico, destacando pontos importantes e orientando o leitor com títulos, legendas e anotações visuais.

🔍 Explicação passo a passo

  • ax.set_title(...): define o título principal do gráfico.
  • ax.set_xlabel(...)/set_ylabel(...): adiciona rótulos descritivos aos eixos X e Y.
  • ax.annotate(...): posiciona texto em xytext, aponta para xy com seta configurada em arrowprops.
  • marker='o', linestyle='-': plota pontos e conecta com linha contínua.
  • plt.show(): renderiza a figura final.

🧾 Exemplo de Saída

Gráfico de série temporal com título “Vendas Mensais”, rótulos nos eixos e uma seta apontando para o pico de vendas em maio.

🗺️ Grades e Ticks

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar figura e eixos
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# 2. Ativar grade principal com linha pontilhada
ax.grid(True, which='major', linestyle=':')

# 3. Definir ticks no eixo X para meses
meses = ['Jan','Fev','Mar','Abr','Mai','Jun','Jul','Ago','Set','Out','Nov','Dez']
ax.set_xticks(range(1, 13))
ax.set_xticklabels(meses, rotation=45)

# 4. Exemplo de plot
x = range(1, 13)
y = [8000, 9500, 12000, 14000, 15000, 13000, 16000, 17000, 15500, 14500, 13500, 12500]
ax.plot(x, y, marker='o')

# 5. Exibir
plt.show()

🧠 Para que serve?

Melhora a legibilidade do gráfico, ajudando a guiar o olhar com linhas de grade e posicionando ticks e rótulos em locais-chave.

🔍 Explicação passo a passo

  • ax.grid(True, which='major', linestyle=':'): exibe apenas as grades principais como linhas pontilhadas.
  • ax.set_xticks(range(1,13)): define posições exatas dos ticks no eixo X.
  • ax.set_xticklabels(meses, rotation=45): atribui rótulos customizados para cada tick e rotaciona para 45°.
  • ax.plot(x, y, marker='o'): plota a série de dados com marcadores circulares.
  • plt.show(): renderiza o gráfico com grades e ticks configurados.

🧾 Exemplo de Saída

Gráfico de linha com grade pontilhada, ticks para cada mês rotacionados a 45°, melhorando a leitura das datas.

🖌️ Mapas de Cor (Colormaps)

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar figura e eixos
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# 2. Dados de exemplo
x = df['quantidade']
y = df['valor_total']

# 3. Scatter com colormap
sc = ax.scatter(x, y, c=df['preco_unitario'], cmap='viridis')
fig.colorbar(sc, ax=ax, label='Preço Unitário')

# 4. Título e rótulos
ax.set_title('Quantidade vs. Valor Total com Colormap')
ax.set_xlabel('Quantidade')
ax.set_ylabel('Valor Total')

# 5. Exibir
plt.show()

🧠 Para que serve?

Permite mapear uma terceira dimensão de dados (aqui, preço unitário) usando uma escala de cores, ideal para scatter plots e heatmaps que revelam padrões e densidades adicionais.

🔍 Explicação passo a passo

  • fig, ax = plt.subplots(...): cria a figura e os eixos para plotagem.
  • ax.scatter(x, y, c=..., cmap='viridis'): plota pontos em (x,y) pintando cada ponto de acordo com o valor de preco_unitario usando o colormap “viridis”.
  • fig.colorbar(sc, ax=ax, label='Preço Unitário'): adiciona uma barra de cores ao lado indicando a escala de valores.
  • ax.set_title/xlabel/ylabel: define título e rótulos dos eixos para contextualizar o gráfico.
  • plt.show(): renderiza a figura completa.

🧾 Exemplo de Saída

Scatter plot onde cada ponto representa uma venda, colorido conforme o preço unitário, com legenda de cores ao lado indicando a intensidade do valor.

🕰️ Gráfico de Série Temporal

🐍 Exemplo Python

import pandas as pd
import matplotlib.pyplot as plt

# 1. Carregar dados e converter para datetime
df = pd.read_csv('vendas.csv')
df['data_venda'] = pd.to_datetime(df['data_venda'], dayfirst=True)

# 2. Criar figura e eixo
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)

# 3. Plotar série temporal com pandas (usa matplotlib internamente)
df.set_index('data_venda')['valor_total'].plot(ax=ax)

# 4. Personalizar título e rótulos
ax.set_title('Vendas ao Longo do Tempo')
ax.set_xlabel('Data de Venda')
ax.set_ylabel('Valor Total (R$)')

# 5. Exibir
plt.show()

🧠 Para que serve?

Permite visualizar a evolução de uma métrica (como faturamento) ao longo do tempo de forma rápida e com formatação automática de datas.

🔍 Explicação passo a passo

  • pd.to_datetime(...): transforma a coluna de strings em objetos datetime.
  • df.set_index('data_venda'): define a data como índice para o plot automático.
  • ['valor_total'].plot(ax=ax): chama o método de plotagem do pandas sobre o eixo matplotlib.
  • ax.set_title/xlabel/ylabel: adiciona contexto ao gráfico com títulos e rótulos.
  • plt.show(): renderiza a figura completa.

🧾 Exemplo de Saída

Gráfico de linha contínua mostrando o valor total de vendas por data, com o eixo X formatado para exibir datas corretamente.

💾 Salvar Figuras

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Criar figura e eixo
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)
ax.plot(df['data'], df['vendas'], marker='o')
ax.set_title('Vendas Mensais')
ax.set_xlabel('Mês')
ax.set_ylabel('Faturamento (R$)')

# 2. Salvar figura em arquivo PNG
fig.savefig('vendas_mensais.png', bbox_inches='tight', dpi=150)

🧠 Para que serve?

Exporta a figura criada para um arquivo de imagem, permitindo incorporá-la em relatórios, apresentações ou sites com controle de resolução e margens.

🔍 Explicação passo a passo

  • fig.savefig('vendas_mensais.png'): especifica o nome e formato do arquivo de saída.
  • bbox_inches='tight': remove espaços em branco extras ao redor da figura.
  • dpi=150: define a resolução em 150 pontos por polegada, gerando imagem mais nítida.

🧾 Exemplo de Saída

Arquivo vendas_mensais.png gerado em alta resolução, com as margens ajustadas ao conteúdo do gráfico.

🎨 Estilos Pré-definidos

🐍 Exemplo Python

import matplotlib.pyplot as plt

# 1. Escolher tema global
plt.style.use('seaborn-darkgrid')  # ou 'ggplot', 'bmh', 'fivethirtyeight', etc.

# 2. Criar figura e eixos com o tema aplicado
fig, ax = plt.subplots(figsize=(10, 6), dpi=100)
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]
ax.plot(x, y, marker='o')
ax.set_title('Gráfico com Estilo Seaborn Darkgrid')
ax.set_xlabel('Eixo X')
ax.set_ylabel('Eixo Y')

# 3. Exibir
plt.show()

🧠 Para que serve?

Aplica um tema visual consistente a todos os gráficos, alterando cores, grades e estilos de linha de forma uniforme para uma apresentação mais profissional.

🔍 Explicação passo a passo

  • plt.style.use('seaborn-darkgrid'): define o estilo global antes de qualquer plotagem.
  • 'seaborn-darkgrid': estilo com fundo escuro e grades claras; outras opções incluem 'ggplot', 'bmh', 'fivethirtyeight'.
  • Após configurar o estilo, fig, ax = plt.subplots(...) cria a figura já estilizada.
  • Os comandos de plotagem (ax.plot, ax.set_title, etc.) usam automaticamente as cores e fontes do tema.
  • plt.show(): renderiza a figura com o tema aplicado.

🧾 Exemplo de Saída

Gráfico de linha com o estilo “seaborn-darkgrid”, exibindo background cinza escuro e grades pontilhadas claras.

🎞️ Animações Simples

🐍 Exemplo Python

from matplotlib.animation import FuncAnimation
import matplotlib.pyplot as plt
import numpy as np

# 1. Criar figura e linha vazia
fig, ax = plt.subplots()
line, = ax.plot([], [], lw=2)

# 2. Função de inicialização
def init():
    ax.set_xlim(0, 10)
    ax.set_ylim(0, 100)
    return line,

# 3. Função de atualização por frame
def update(frame):
    x = np.linspace(0, 10, 100)
    y = np.sin(x + frame/10) * 50 + 50
    line.set_data(x, y)
    return line,

# 4. Criar animação
anim = FuncAnimation(
    fig, update, init_func=init,
    frames=100, interval=50
)

# 5. Salvar como GIF
anim.save('animacao.gif', writer='imagemagick')

🧠 Para que serve?

Permite criar animações de gráficos, ideal para visualizar a evolução dinâmica de dados ao longo do tempo ou demonstrar comportamentos periódicos.

🔍 Explicação passo a passo

  • FuncAnimation: gera a animação, chamando update para cada frame e usando init para configurar os eixos inicialmente.
  • init(): define limites do eixo X (0–10) e Y (0–100) antes da primeira atualização.
  • update(frame): recalcula os dados de x e y para cada quadro, criando um efeito de onda senoidal em movimento.
  • frames=100, interval=50: especifica 100 quadros com intervalo de 50 ms entre eles, controlando a duração e a fluidez.
  • anim.save(...): exporta a animação para um arquivo GIF usando o writer ImageMagick.

🧾 Exemplo de Saída

Arquivo animacao.gif mostrando uma onda senoidal animada que oscila entre 0 e 100 no eixo vertical.

🔧 Customização Avançada

🐍 Exemplo Python

import matplotlib.pyplot as plt
from mpl_toolkits.axes_grid1.inset_locator import inset_axes
from matplotlib.patches import ConnectionPatch
import matplotlib.gridspec as gridspec
import numpy as np

# Dados de exemplo
x = np.linspace(0, 10, 100)
y1 = np.sin(x)
y2 = np.cos(x)

# 1. Eixos secundários
fig, ax1 = plt.subplots(figsize=(10, 6), dpi=100)
ax1.plot(x, y1, color='b', label='Seno')
ax1.set_xlabel('X')
ax1.set_ylabel('sin(x)', color='b')
ax1.tick_params(axis='y', labelcolor='b')
ax2 = ax1.twinx()
ax2.plot(x, y2, color='r', linestyle='--', label='Cosseno')
ax2.set_ylabel('cos(x)', color='r')
ax2.tick_params(axis='y', labelcolor='r')

# 2. Inset plot (zoom no seno)
axins = inset_axes(ax1, width="30%", height="30%", loc='upper right')
axins.plot(x, y1, color='green')
axins.set_title('Zoom Seno')

# 3. Broken axis com gridspec
fig2 = plt.figure(figsize=(6, 4), dpi=100)
gs = gridspec.GridSpec(2, 1, height_ratios=[1, 2])
ax_top = fig2.add_subplot(gs[0])
ax_bottom = fig2.add_subplot(gs[1], sharex=ax_top)
ax_top.plot(x, y1)
ax_bottom.plot(x, y1)
ax_top.spines['bottom'].set_visible(False)
ax_bottom.spines['top'].set_visible(False)
ax_top.tick_params(labelbottom=False)
con = ConnectionPatch(
    xyA=(0,1), xyB=(0,0),
    coordsA="axes fraction", coordsB="axes fraction",
    axesA=ax_top, axesB=ax_bottom,
    arrowstyle='-'
)
ax_bottom.add_artist(con)

# 4. Textos em LaTeX
plt.rc('text', usetex=True)
fig3, ax3 = plt.subplots(figsize=(6, 3), dpi=100)
ax3.set_title(r'$\alpha > \beta$', fontsize=14)
ax3.plot(x, y1, label=r'$\sin(x)$')
ax3.legend()

plt.show()

🧠 Para que serve?

Combina técnicas avançadas para enriquecer visualizações: eixos duplos, zoom embutido, eixos quebrados, layouts flexíveis e textos matemáticos em LaTeX.

🔍 Explicação passo a passo

  • ax1.twinx(): cria um segundo eixo Y compartilhando o X, útil para comparar séries com escalas diferentes.
  • inset_axes(...): adiciona um gráfico menor dentro do principal, perfeito para destacar regiões de interesse.
  • gridspec.GridSpec: configura uma grade para criar “eixos quebrados” com proporções customizadas.
  • ConnectionPatch: desenha conexões visuais entre eixos quebrados, mantendo a continuidade de dados.
  • plt.rc('text', usetex=True): habilita o uso de sintaxe LaTeX para títulos, rótulos e anotações matemáticas.
